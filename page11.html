Video Project - Storyboard/Plan

Plan:

For this project I would like to insert myself into multiple different films, as a dream that I have while watching movies. I think it would be really funny to throw myself into a couple Tarantino flicks but I'm not exclusively set on that. For the animation part, I'm unsure, but I had the thought of replicating the animation from Kill Bill Vol 1: https://www.youtube.com/watch?v=KQM0klOXck8. I don't know how realistic this is since I don't have any experience in animation. 

I'm planning to test out a couple different scenes with my green screen set up before I choose what to do specifically, since this project requires a bit more pre-production planning. The story boards for those selected scenes follow the actually scenes of the films, so that's why I don't have a complete storyboard. The end of my video should follow the beginning format, with me waking up in a sweat, then with movie credits on the TV.
